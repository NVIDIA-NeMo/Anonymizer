    Getting Started¶

    This guide walks you through installing Anonymizer, configuring model providers, and running your first anonymization.

    Prerequisites¶

    • Python 3.11+
    • An NVIDIA API key (free tier available) for default model providers

    Install¶

    git clone https://github.com/NVIDIA/Anonymizer.git
cd Anonymizer
make install

    git clone https://github.com/NVIDIA/Anonymizer.git
cd Anonymizer
pip install -e .

    Verify the installation:

    anonymizer --help

    Set up model providers¶

    Anonymizer uses models hosted on build.nvidia.com by default -- GLiNER-PII for entity detection and a text LLM for augmentation, validation, and replacement.

    export NVIDIA_API_KEY="your-nvidia-api-key"

    Custom model endpoints

    You can bring your own models (vLLM, OpenAI, etc.) via provider configs. See Models for details.

    Your first anonymization¶

    Option A: CLI¶

    Preview redaction on a small sample:

    anonymizer preview --source data.csv --replace redact

    Run the full pipeline and write output:

    anonymizer run --source data.csv --replace redact --output result.csv

    Validate your config without processing any data:

    anonymizer validate --source data.csv --replace hash

    Run anonymizer --help or anonymizer <subcommand> --help for all options.

    Option B: Python API¶

    from anonymizer import Anonymizer, AnonymizerConfig, AnonymizerInput, Redact

anonymizer = Anonymizer()

config = AnonymizerConfig(replace=Redact())
data = AnonymizerInput(source="data.csv", text_column="text")

# Preview on a small sample
preview = anonymizer.preview(config=config, data=data, num_records=3)

    Inspect results¶

    # Interactive visualization with entity highlights and replacement map
preview.display_record()

# User-facing columns (original text + replaced text + entities)
preview.dataframe

# Full pipeline trace with all internal columns
preview.trace_dataframe

    display_record() renders an HTML visualization in notebooks showing the original text with highlighted entities, the replacement map, and the anonymized output. Calling it repeatedly cycles through records.

    Full run¶

    result = anonymizer.run(config=config, data=data)

# result.dataframe      -- output dataframe
# result.trace_dataframe -- full trace
# result.failed_records  -- any records that failed processing

    Try a different strategy¶

    Anonymizer supports four replacement strategies:

    from anonymizer import Redact, Annotate, Hash, Substitute

# Redact: replace with a label marker
AnonymizerConfig(replace=Redact())
# "Alice" -> "[REDACTED_FIRST_NAME]"

# Annotate: tag entities with their label
AnonymizerConfig(replace=Annotate())
# "Alice" -> "<Alice, first_name>"

# Hash: deterministic hash token
AnonymizerConfig(replace=Hash())
# "Alice" -> "<HASH_FIRST_NAME_3bc51062973c>"

# Substitute: LLM-generated synthetic replacement
AnonymizerConfig(replace=Substitute())
# "Alice" -> "Maya"

    See Replace Strategies for customization options and a comparison guide.

    Rewrite mode¶

    Instead of replacing individual entities, rewrite mode generates a privacy-safe paraphrase of the entire text:

    from anonymizer import Anonymizer, AnonymizerConfig, AnonymizerInput, Rewrite

anonymizer = Anonymizer()

config = AnonymizerConfig(rewrite=Rewrite())
data = AnonymizerInput(source="data.csv", text_column="text")

preview = anonymizer.preview(config=config, data=data, num_records=3)
preview.display_record()

    The result includes utility_score, leakage_mass, and a needs_human_review flag. See Rewrite Mode for details.
